# KR 시세를 통합(SOR `_AL`) 코드로 — NXT 확장시간 포함 통합시세 채택

키움에서 KR 종목 시세(현재가 ka10001, 일/주/월봉 ka10081/82/83)를 **통합(SOR) 거래소 코드(`{종목}_AL`)**로 조회한다(`client.integrated_code()`가 맨코드에 `_AL` 부착). 이렇게 하면 가격이 **KRX + 넥스트레이드(NXT)의 합성 BBO 통합호가**가 되어, KRX 정규장(09:00~15:30) 밖의 NXT 프리/애프터마켓(08:00~20:00) 가격까지 포착된다. 자동폴링 윈도우(`isKrMarketOpen`)도 08:00~20:00로 맞췄다. 라이브 프로브에서 **`_AL`이 전 종목 안전**함을 확인했다 — NXT 거래종목은 통합가(예 005930 KRX 322,500 vs 통합 324,500), 비-NXT 종목은 통합이 KRX값으로 자동 폴백(별도 멤버십 체크 불필요).

## Considered Options

- **맨 KRX 코드 유지(기존)** — 거부. NXT 확장시간(08:00~09:00, 15:30~20:00) 가격을 못 잡는다. 사용자가 확장시간 실시간을 원했다.
- **`_NX`(NXT 전용) 코드** — 거부. 비-NXT 종목은 빈 응답을 주고(NXT는 ~650종목 subset), 차트 히스토리도 짧다(2025-03 출범, ~298봉). 전 종목 일관 처리 불가.
- **종목별 NXT 멤버십 체크 후 분기** — 불필요. 프로브에서 `_AL`이 비-NXT도 KRX로 자동 폴백함을 확인 → 단일 코드로 충분.

## Consequences

- **표시 가격이 KRX→통합으로 이동한다**(005930 322,500→324,500). 더 최신·확장시간 포함이지만, Naver/KRX 단독값과 다를 수 있다(의도된 변화).
- **통합호가는 합성 BBO라 호가창에서 시세역전(매수호가>매도호가)이 정상 발생**할 수 있다(두 거래소 독립 가격형성). 우리는 현재가(cur_prc)·OHLC만 쓰므로 호가창 미사용 — 영향 적지만, 향후 호가 표시 시 유의.
- **수급(ka10059)·외국인 보유율(ka10008)은 통합 적용 안 함** — 시세가 아니라 KRX 기반 집계라 맨코드 유지.
- 시세 정본은 여전히 키움 우선 + 기존소스(Naver/yfinance) 폴백(ADR-0009). `_AL` 실패 시에도 폴백 경로가 받는다.
- ⚠️ NXT 세션·종목범위·시간은 2025 출범 후 변동 중(딥리서치 2026-02 기준 ~650종목). 확장시간 실가격 갱신은 평일 확장세션에서 관측 필요.
